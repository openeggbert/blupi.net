Welcome to our comprehensive list of games featuring Blupi, also known as Speedy Eggbert. Created by Daniel Roux, these games span various genres including educational, puzzle, and strategy. This collection showcases the diversity and charm of Blupi games, highlighting the creativity and innovation behind each title.

For additional information and to explore a wider range of Blupi-related games, visit the Blupi Fandom Games Category.

Complete List of Blupi Games

  • Toto a la maison (1988) - Early Blupi game featuring family-oriented content.
  • Toto s'amuse (1990) - A game focused on entertainment and fun.
  • Toto Blupi at Home - An educational game designed for learning.
  • Toto a la campagne - Adventure game set in the countryside.
  • Fun with Blupi - Interactive game with various mini-games.
  • Blupi's Travels - A travel-themed game featuring Blupi.
  • Toto apprend le vocabulaire - Vocabulary learning game.
  • Blupi Explorer (1993) - Exploration game with educational elements.
  • Blupimania (1994) - A challenging puzzle game.
  • Planet Blupi (1997) - Real-Time Strategy game set on a fictional planet.
  • Speedy Blupi I (1998) - A popular platformer game.
  • Speedy Eggbert 1 - The first entry in the Speedy Eggbert series.
  • Speedy Blupi II (2001) - Sequel to Speedy Blupi I.
  • Speedy Eggbert 2 - Follow-up to Speedy Eggbert 1.
  • CoLoBot (2001) - Educational game teaching programming basics.
  • Ceebot (2002) - Programming-focused game.
  • Buzzing Cars (2002) - Racing and driving game.
  • Blupimania II (2003) - Second installment in the Blupimania puzzle series.
  • Speedy Blupi for Windows Phone (2013) - Mobile adaptation of Speedy Blupi.
  • BlupiEdit - Tool for editing Blupi-related content.
  • tiled-blupi - Game development tool for Blupi projects.
  • X-Edit - Editing software for Blupi games.
  • X-Edit-Mapper - Mapping tool for Blupi games.
  • Speedy Eggbert - Godot prototype - Video Demo - Prototype demonstration video.
  • Blupi is Back - Latest project reviving the Blupi character.
  • Eggbert 3 - Upcoming game in the Eggbert series.
  • Open Eggbert (2024 - development started) - New open-source project in development.
